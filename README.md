## Script Descriptions

## Script 1 — System Identity Report

This script displays basic system details like kernel version, logged-in user, uptime, and current date.

## Script 2 — Python Package Inspector

This script checks whether Python is installed and shows its version along with a short description.

## Script 3 — Disk and Permission Auditor

This script analyzes important system directories and displays their permissions and disk usage.

## Script 4 — Log File Analyzer

This script reads a log file and counts how many times a keyword (default: "error") appears.

## Script 5 — Open Source Manifesto Generator

This script takes user input and generates a personalized open-source manifesto, which is saved in a text file.
